# 🦐 Shrimps-cn

A small React component library built with shadcn/ui, Tailwind CSS, and TypeScript. Install components directly into your project and own the code — no black boxes.

## CLI

Shrimps has a dedicated CLI for adding components and setting up the CSS theme.

```bash
# npm
npx shrimps init
npx shrimps add button

# pnpm
pnpm dlx shrimps init
pnpm dlx shrimps add button
```

## Philosophy

No npm package to install and keep updated. Each component is copied straight into your codebase — yours to read, tweak, and own.
