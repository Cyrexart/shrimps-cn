# 🦐 Shrimps-cn

A small React component library built with shadcn/ui, Tailwind CSS, and TypeScript. Install components directly into your project and own the code — no black boxes.

## Installation

Requires [shadcn/ui](https://ui.shadcn.com/docs/installation) to be set up in your project.

Then add any component via:

```bash
npx shadcn add Cyrexart/shrimps-cn/<component>
```

For example:

```bash
npx shadcn add Cyrexart/shrimps-cn/button
```

## Philosophy

No npm package to install and keep updated. Each component is copied straight into your codebase — yours to read, tweak, and own.

## Tech

- React + TypeScript
- Tailwind CSS
- shadcn/ui / Radix UI primitives
